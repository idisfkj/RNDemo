# RNDemo
基于`React Native` 一个`App`，这里只是对`Android`部分进行了适配,后续会继续扩展。详情说明请移步[idisfkj](http://idisfkj.github.io/2016/08/29/React-Native-For-Android/)

# 安装步骤
1.第一步

```
git clone https://github.com/idisfkj/RNDemo.git
```

2.第二步

```
cd RNDemo
```

3.第三步

```
npm install
```

4.第四步

Mac
```
react-native run-android
```

Windows

```
react-native start
react-native run-android
```
